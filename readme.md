# Sinatra Hello World Example
This an example of a basic hello world application for Sinatra. 

# How To Run

Install Your Gems

`gem install sinatra`

`gem install activerecord`

`gem install thin`

Download the Files

`git clone git@github.com:nickbarth/Sinatra-Hello-World.git`

`cd Sinatra-Hello-World`

Start the server

`thin start`
